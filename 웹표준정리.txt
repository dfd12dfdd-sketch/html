★★★
ul li {
   list-style: none; /* li 블릿 (원형 기호) 없애는 태그
   box-sizing: border-box; /* 테두리, 패딩 적용없이 정사이즈 (박스크기 수치대로 반영) */
}

- border: solid 1px 색상코드; => 박스 선 넣기
  (border-bottom: solid 1px black; 밑줄만 -> border-top & border-left & border-right 응용가능)

- border-radius: 20px; / border-radius: 50%; 원형 / => 모서리 곡선
- box-sizing: border-box; => 테두리, 패딩 적용없이 정사이즈 (박스크기 수치대로 반영)

- margin: 0 auto; => 상하값 0, 좌우값 auto가 전체 가운데 배치하는 태그
- p:nth-child(1) => 1줄만 적용
- item > span => 선택자 > 는 자식선택자 선택, > 없으면 자식,자손 모두 선택(전부 선택됨) (ex) > 입력 시 span 1곳만 적용O
- line-height: 1; => line height 숫자값은 font size의 배수 80*1=>80 (6일차예제 참고. 프로덕트 폰트들이 상단배치)

# 라운딩 버튼 예제문 
- text-decoration: none; => 텍스트 폰트 밑줄 등 없애기
- display: block; => 가로세로 사이즈 입력가능케 함 (인라인 요소인 a 태그 사이즈 설정하기)
- line-height: 35px; => 버튼 상하 가운데 배치
  height: 35px; (한 세트 필수! 35px -> height 35와 동일해야 함)
